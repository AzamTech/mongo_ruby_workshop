# MongoDB + Ruby Workshop

To get started, make sure that MongoDB 2.x is installed and running, and run:

```bash
bundle install
bundle exec guard
```

You can also check out the [complete](https://github.com/afeld/mongo_ruby_workshop/tree/complete) demo.
